## 📋 项目概述

这是一个基于Tushare财经数据平台 + 硅基流动Deepseek-V3 AI的**股票夜间持仓分析系统**。

**核心功能**：
- 📊 获取股票实时数据（基本面、日线、5分钟线、资金流向）
- 🤖 AI智能分析（基于多维度数据的量化评分）
- 📈 生成分析报告并上传OSS云存储
- ⏰ 支持定时采集和阿里云FC函数计算部署

**特色**：
- ✅ 支持特定股票分析（301117等问题股票已通过测试）
- ✅ 5分钟K线实盘数据（新浪API双URL容错方案）
- ✅ 完整的资金流向分析（近5日主力资金动向）
- ✅ AI分析包含6个维度评估

### 1. data_collector.py - 数据采集器

**主要类**：`TushareCollector`

#### 关键方法

**获取5分钟K线数据**
```python
from data_collector import TushareCollector
```

返回格式：
```python
[
    {
        'time': '2025-12-10 14:45:00',
        'open': 51.61,
        'high': 52.0,
        'low': 51.6,
        'close': 51.94,
        'vol': 461572
    },
    ...
]
```

**获取完整股票详情**
```python
# 方法2：获取完整数据（推荐）
stock_detail = collector.get_stock_detail('301117', '20251210')

# 返回包含以下字段：
# - code: 股票代码
# - name: 股票名称
# - price: 当前价格
# - minute_data: 今日5分钟线 (100条)
# - daily_data: 近5日历史数据
# - capital_flow: 5日资金流向分析
# - pe, pb: 估值指标
```

#### 实现细节

**5分钟线获取（新浪API双URL容错方案）**：
- URL1（主）：标准URL
- URL2（备）：带时间戳参数的备选URL
- 自动转换：若URL1失败，自动切换URL2
- 多格式支持：同时支持对象格式和数组格式响应

**代码示例**（无需调用，已内置）：
```python
def get_sina_minute_data(self, code: str) -> list:
    # 尝试URL1
    try:
        resp = requests.get(url1, params=params, timeout=5)
        minute_data = self._parse_sina_minute_data(resp.text, sina_code)
        if minute_data:
            return minute_data
    except:
        pass
    
    # URL1失败，尝试URL2
    try:
        resp = requests.get(url2, params=params, timeout=5)
        minute_data = self._parse_sina_minute_data(resp.text, sina_code)
        if minute_data:
            return minute_data
    except:
        pass
    
    return []  # 两个URL都失败
```

### 2. ai_analyzer.py - AI分析引擎

**主要类**：`AIAnalyzer`

#### 使用方法
```python
from ai_analyzer import AIAnalyzer
import asyncio

async def analyze():
    analyzer = AIAnalyzer()
    stock_data = collector.get_stock_detail('301117', '20251210')
    
    result = await analyzer.analyze_for_overnight(stock_data)
    
    print(f"评分: {result['score']}")
    print(f"推荐: {result['recommendation']}")
    print(f"分析:\n{result['analysis']}")

asyncio.run(analyze())
```

#### AI分析6个维度

1. **尾盘强度** - 基于日线数据
   - 成交额、量比、涨幅的综合判断

2. **5分钟K线节奏** - 基于分钟线数据 ✨ NEW
   - 是否有突破新高？
   - 是否出现加速上涨？
   - 成交量是否温和放大？

3. **主力意图** - 基于5日资金流向
   - 低位吸筹还是高位出货？

4. **隔夜风险** - 风险评估
   - 是否有明显出货迹象？
   - 成交额是否萎缩？

5. **次日预判** - 基于历史数据
   - 预测次日走势

6. **操作建议** - 实战指导
   - 具体的买入价、止损位、目标位

#### 返回格式
```python
{
    'score': 85,  # 0-100分
    'recommendation': '强烈推荐',  # 强烈推荐/推荐/持观望/不建议
    'analysis': 'AI的详细分析文本'
}
```

## ⚙️ 高级用法

### 1. 自定义股票分析

```python
from data_collector import TushareCollector
from ai_analyzer import AIAnalyzer
from report_generator import ReportGenerator
import asyncio

async def analyze_custom(code, date):
    collector = TushareCollector()
    analyzer = AIAnalyzer()
    generator = ReportGenerator()
    
    # 1. 获取数据
    stock_detail = collector.get_stock_detail(code, date)
    
    # 2. AI分析
    ai_result = await analyzer.analyze_for_overnight(stock_detail)
    
    # 3. 生成报告
    html = generator.generate_html_report(stock_detail, ai_result)
    
    # 4. 上传OSS（可选）
    from oss_uploader import OSSUploader
    uploader = OSSUploader()
    uploader.upload_file_content(
        content=html,
        remote_path=f'reports/{code}_{date}.html'
    )
    
    return ai_result

# 运行
result = asyncio.run(analyze_custom('301117', '20251210'))
print(result)
```

### 2. 定时采集（本地）

```python
from scheduler import DailyScheduler

scheduler = DailyScheduler()

# 每日14:30执行
scheduler.schedule_daily('14:30', analyze_custom)
scheduler.start()
```
